# AI×MUS｜筋トレ管理×AIトレーナー

トレーニング記録、目標別メニュー、AIフォーム解析を統合する筋トレ管理Webアプリです。

## 公開サイト

- スマホ・PC共通: https://aimus-training.onrender.com/
- ヘルスチェック: https://aimus-training.onrender.com/api/v1/health
- Render管理画面: https://dashboard.render.com/web/srv-d8pkejugvqtc7399hjm0

現在はRender無料プランで公開しています。しばらくアクセスがない場合はスリープし、再起動に時間がかかることがあります。また、永続ディスクがないため、再デプロイ時に登録データや動画が消える可能性があります。

常時稼働・データ永続化には `render.always-on.yaml` の有料構成を使用します。
